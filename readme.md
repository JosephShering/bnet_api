# Battle Net GraphQL
This project's intended purpose is to create a translation from the BattleNET API to a GraphQL API. We are still in active development.


## Getting Started
```
yarn install
yarn start
```

Open your browser and navigate to [localhost](https://localhost:4000)

## Running Tests
```
yarn test
```
